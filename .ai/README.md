# VERDEA AI Context

Esta carpeta contiene las reglas operativas para agentes de IA y colaboradores que utilicen herramientas de vibecoding.

## Documento obligatorio

- [AI Development Charter](AI_DEVELOPMENT_CHARTER.md)

## Flujo antes de generar código

1. Leer `.foundation/`.
2. Leer este directorio.
3. Leer la SPEC del dominio.
4. Inspeccionar componentes y contratos existentes.
5. Proponer un plan cuando el cambio sea Nivel 1 o 2.
6. Implementar el cambio mínimo necesario.
7. Ejecutar pruebas y actualizar documentación.

## Principio central

El chat o prompt no es una fuente de verdad. Todo contexto necesario para mantener el cambio debe quedar dentro del repositorio.
